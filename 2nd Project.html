Life Club Mixed Martial Arts

Project Description

This website simulates a neighborhood Mixed Martial Arts (MMA) academy. It includes a class registration form and an online store with MMA-related products, integrating a functional payment gateway. The design aims to convey the strength and discipline of MMA, with a modern and sporty aesthetic.

Video Demonstration

Technologies Used

  • HTML5 and CSS3 for structure and design
  • JavaScript for interactivity
  • Simulated payment gateway
  • Responsive design for mobile and desktop
